# Introducción

En la actualidad, las nuevas tecnologías forman parte de la vida cotidiana. Cada día se utilizan plataformas digitales para estudiar, trabajar, realizar transacciones bancarias y almacenar información personal. Debido a ello, la seguridad informática se ha convertido en una necesidad fundamental.

Este proyecto consiste en el desarrollo de un Generador Seguro de Contraseñas en Python, el cual permite crear contraseñas robustas y difíciles de vulnerar, contribuyendo a proteger la información de los usuarios.

---

# Descripción del problema

Muchas personas utilizan contraseñas débiles como nombres, fechas de nacimiento o secuencias numéricas fáciles de adivinar, lo que incrementa el riesgo de ataques informáticos.

Este sistema busca solucionar ese problema permitiendo generar contraseñas aleatorias con distintos niveles de complejidad.

---

# Objetivo del sistema

Desarrollar un software funcional que permita generar contraseñas seguras utilizando Python, aplicando los conocimientos adquiridos durante la asignatura de Lógica de Programación.

---

# Funcionalidades del sistema

- Generación automática de contraseñas.
- Selección de longitud personalizada.
- Inclusión de letras mayúsculas.
- Inclusión de números.
- Inclusión de símbolos especiales.
- Validación de longitud mínima.
- Evaluación del nivel de seguridad (Baja, Media o Alta).

# Explicación del sistema

El programa solicita al usuario la longitud deseada para la contraseña y pregunta si desea incluir letras mayúsculas, números y símbolos especiales.

Con esta información genera una contraseña aleatoria utilizando las librerías `random` y `string`.

Posteriormente analiza la contraseña y determina su nivel de seguridad como:

- Baja
- Media
- Alta
